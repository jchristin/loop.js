loop.js by julbaxter

The same way that CSS provides style, loop.js brings the behavior

What is it ?

Loop.js is another way to put javascript code in web applications. It allow to bind pieces of code to HTML elements via CSS selectors. The whole is then executed each loop iteration.

It turns the traditional event-driven programming in loop based programming, which is more confortable and more suitable for dynamic websites.

How to use it ?

1 - Create a .loop file as you would for CSS and name it "whattyouwant.loop".
2 - Write some javascript code in CSS selectors.
3 - Link it as a CSS file.
4 - include loop.js and the end of the body element.

Example

index.html

<head>
<link rel="loop" type="text/javascript" href="behavior.loop">
</head>
<body>
<table id="dock">
<tr>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
<td><div class="icon"></div></td>
</tr>
</table>
<script src="loop.js" type="text/javascript"></script>
</body>

behavior.loop

.icon
{
var Size = 50;
var Opacity = 0.5;

var Dock = document.getElementById( "dock" );
if( IsMouseInside( Dock.parentNode ) )
{
var Dist = Math.min( 200, Math.abs( MouseX - GetOffsetLeft( This ) ) );
Size = InterpolateEx( 50, 100, 200, 0, Dist);
Opacity = InterpolateEx( 0.5, 1, 200, 0, Dist);
}

This.style.width = "" + Size + "px";
This.style.height = "" + Size + "px";
This.style.opacity = Opacity;
}
